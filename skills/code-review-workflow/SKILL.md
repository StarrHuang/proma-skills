# Code Review Workflow

## Requesting Review
完成功能实现/重大修改/合并前，请求他人审查以验证工作满足需求。

## Receiving Review
收到反馈后：① 理解每条建议的意图 ② 独立验证技术正确性 ③ 有疑问时追问而非盲从 ④ 逐条acknowledge并实施或反驳。
